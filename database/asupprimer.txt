1 - Cloner le repo :

git clone https://github.com/OlivierBldn/herochatting.git

Optionnel pour copier le projet sur votre github et le modifier à votre guise :

1 - Créer un nouveau repository github
2 – Ouvrir un terminal dans le dossier du projet que vous venez de cloner (qui porte le nom herochatting si vous ne l’avez pas modifié)
3 – Changer l’url du repository pour celle du repository que vous venez de créer :
git remote set-url origin https://github.com/username/example.git
4 – Envoyer les éléments du projet sur votre repo git :
git push -u origin main
